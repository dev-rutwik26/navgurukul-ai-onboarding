## 📚 Engineering Practices Reading List

### Foundational (Week 1)
- [The Pragmatic Programmer — Key Takeaways](https://pragprog.com/tips) — 20 tips that map directly to our coding standards
- [SOLID Principles Explained](https://www.digitalocean.com/community/conceptual-articles/s-o-l-i-d-the-first-five-principles-of-object-oriented-design) — We follow these
- [How to Write Good Commit Messages](https://cbea.ms/git-commit) — The standard we follow

### Architecture (Week 2)
- [Clean Architecture in Python](https://www.cosmicpython.com) — Free book, highly relevant
- [FastAPI Best Practices](https://github.com/zhanymkanov/fastapi-best-practices) — GitHub repo, practical
- [React Component Patterns](https://www.patterns.dev) — Modern React patterns we use

### AI/ML Engineering (Week 3)
- [ONNX Runtime Web — Browser Inference](https://onnxruntime.ai/docs/tutorials/web/) — How we run models client-side
- [Edge AI Overview](https://www.edge-ai-vision.com/what-is-edge-ai/) — Our architectural philosophy
- [Whisper Paper](https://arxiv.org/abs/2212.04356) — Understanding our STT model
