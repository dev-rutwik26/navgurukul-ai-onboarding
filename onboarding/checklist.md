# Onboarding Checklist

Track your progress through the NavGurukul AI engineering onboarding. Check off each item as you complete it and share your progress with your team lead.

> 💡 **Rule:** Don't rush to check boxes. Each item should be genuinely complete, not just skimmed.

---

## 🟣 Pre-Start (Before Day 1)

- [ ] Received and accepted GitHub organization invitation
- [ ] Created GitHub account with NavGurukul company email
- [ ] Enabled Two-Factor Authentication (2FA) on GitHub
- [ ] Got access to the Notion workspace
- [ ] Read the [Engineering Handbook](../docs/handbook.md) end-to-end

---

## 🔵 Day 1 — Foundation

- [ ] Completed [Environment Setup](../docs/environment-setup.md)
  - [ ] Node.js 18+ installed and verified
  - [ ] Python 3.10+ installed and verified
  - [ ] IDE configured with linting extensions
  - [ ] Git configured with NavGurukul email
- [ ] Received repository access from team lead
- [ ] Successfully cloned assigned repository
- [ ] Created personal branch `dev/your_name`
- [ ] Attended first standup and shared daily intentions
- [ ] Read [Git Workflow guide](../docs/git-workflow.md)
- [ ] Read [Coding Standards](../docs/coding-standards.md)
- [ ] Read [Tech Stack overview](../docs/tech-stack.md)

---

## 🟢 Week 1 — Exercises & First PR

- [ ] Completed [Week 1 Tasks](./tasks/week-1.md)
  - [ ] Exercise 01: Python Basics — all assertions pass
  - [ ] Exercise 03: Git Practice — Tasks 1–5 complete
  - [ ] Opened first PR to `dev`
  - [ ] PR reviewed and merged by team lead

---

## 🟡 Week 2 — First Real Contribution

- [ ] Completed [Week 2 Tasks](./tasks/week-2.md)
  - [ ] Exercise 02: FastAPI Basics — all endpoints working
  - [ ] Exercise 03: Git Practice — reviewed a teammate's PR
  - [ ] Picked up first real task from Notion sprint board
  - [ ] Opened PR for real task with full PR template

---

## 🔴 Week 3 — Independence

- [ ] Completed [Week 3 Tasks](./tasks/week-3.md)
  - [ ] Exercise 04: AI Pipeline — all tests pass
  - [ ] Resolved first merge conflict
  - [ ] Contributed to team sprint planning
  - [ ] Passed team lead onboarding sign-off review

---

## 🎓 Onboarding Complete

- [ ] Team lead has signed off on onboarding completion
- [ ] You have merged at least 3 PRs to `dev`
- [ ] You have reviewed at least 2 teammates' PRs
- [ ] You have shared your daily intentions in standup every day

---

> **Congratulations!** Once the final section is complete, you're a fully onboarded NavGurukul AI engineer. 🚀
