# Exercise 01 — Python Basics & Clean Code

## 🎯 Goal
Practice writing **clean, modular, well-commented Python** — the standard expected in all NavGurukul codebases.

## 📋 Instructions
1. Open `task.py`
2. Complete every function marked with `# TODO:`
3. Do **not** change function signatures — only fill in the bodies
4. Add docstrings and inline comments explaining your logic
5. Run the file: `python task.py` — all assertions must pass
6. Commit and open a PR to `dev`

## ✅ Acceptance Criteria
- [ ] All `# TODO:` items are completed
- [ ] All built-in `assert` statements pass when running `python task.py`
- [ ] Every function has a proper docstring
- [ ] Code is modular — no function exceeds 20 lines
- [ ] No hardcoded magic numbers (use named constants)
- [ ] PR is opened against `dev` with this exercise linked

## 💡 Hints
- Use list comprehensions where appropriate
- Python's `statistics` module may be useful
- Think about edge cases: empty lists, negative numbers, zero
